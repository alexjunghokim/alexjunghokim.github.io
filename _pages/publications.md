## Publications

- **Automatic App Image Generation and Selection**
  First Author. Apple Machine Learning Summit, January 2024.
  Automated image generation and editing system using diffusion models and vision-language scoring for software applications.

- **Automatic Creative Generation with Inpainting**
  First Author. Apple Search Summit 2023 (Full Talk), Apple Machine Learning Summit 2023.
  Text-guided image generation and editing pipeline combining latent diffusion blending with vision-language scoring.

- **Automatic Creative Selection with Cross-Modality Matching**
  First Author. BayLearn 2023 (Main Presentation).
  Cross-modal search relevancy model using late fusion of contextual BERT embeddings and Vision Transformer representations.
